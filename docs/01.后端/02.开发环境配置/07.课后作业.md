## 2.7 课后作业

1. 检查安装的jdk版本，确保是JDK 8，输出其版本号。
2. 检查Maven版本，确保其使用了国内阿里云镜像仓库。
3. 使用图形化管理工具在MySQL中创建一个数据库，创建一张表，并插入两条数据。
4. 验证开发环境，能够正确运行一个最简单的Spring Boot程序。
