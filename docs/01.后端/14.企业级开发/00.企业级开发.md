# 第14章 企业级开发

在企业应用中，有很多和具体业务不相关的必须功能，例如在某些业务的审批节点发送邮件，日终定时调度批处理等。这些通用功能统称为企业级开发功能。

Spring Boot 在企业级开发方面提供或整合了很多便利的基础技术支持框架，如 Spring Batch，Quartz 等。

本章我们学习一些如何通过 Spring Boot 整合常用的企业级开发支持框架，以方便在实际工作中应用。
